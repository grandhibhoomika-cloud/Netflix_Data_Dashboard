# Netflix Data Visualization Dashboard

## Overview
Interactive dashboard built using Streamlit and Python.

## Features
- Dataset Preview
- Total Titles KPI
- Movies and TV Shows Count
- Movies vs TV Shows Chart
- Top 10 Countries Chart
- Release Year Trend
- Ratings Distribution

## Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit

## How to Run
python -m streamlit run app.py
